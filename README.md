# Asom Hindi Bhasi Brahman Mahasabha Website

A complete dynamic organization management website with Supabase backend.

## Features

### User-facing Website
- Home page with latest news, events, and gallery preview
- About Us with mission, vision, and committee
- Member Directory with search and pagination
- Online Membership Registration
- Events & Programmes with registration
- News & Notices with archive
- Photo Gallery with lightbox
- Video Gallery with player
- Donation Page with Razorpay integration
- Documents Section for downloads
- Contact Us with form and map
- Multi-language support (English, Hindi, Assamese)
- Mobile responsive design
- WhatsApp integration
- Digital membership certificates

### Admin Panel
- Secure admin login
- Dashboard with statistics and charts
- Member management with approval system
- Event management with registration tracking
- News management with important notices
- Photo/Video gallery management
- Donation reports with export
- Document management
- Contact message management
- Real-time updates
- Bulk actions
- Export capabilities (PDF, Excel, CSV)

## Tech Stack

- HTML5
- CSS3 (with CSS variables, responsive design)
- Vanilla JavaScript
- Supabase (PostgreSQL, Authentication, Storage)
- Razorpay (Payment Gateway)
- Chart.js (Data visualization)
- jsPDF (PDF generation)
